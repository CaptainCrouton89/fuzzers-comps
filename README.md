## Downloading the Embeddings
Run the following commands
1. `mkdir embedding`
2. `wget http://cs.carleton.edu/faculty/awb/cs208/f21/lab0-handout.tar http://nlp.stanford.edu/data/wordvecs/glove.6B.zip`
3. `unzip glove.6B.zip -d embedding/`
